Simplest Built-in Directives
Transform today date ({{todayDate}}) using 'date' pipe
Transformed today date - {{todayDate | date}}
Transform today date ({{todayDate}}) using 'date' pipe in 'MM/dd/yy' format
Transformed today date - {{todayDate | date : 'MM/dd/yy'}}
Directives Chaining
Transform your money ({{money}}) in Euro currency (first with symbol, next with code) and everything should be in upper case.
Balance: {{money | currency : 'Euro' : 'symbol' | lowercase}}
Balance: {{money | currency : 'USD' : 'code' | uppercase}}
